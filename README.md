# Cadet Connection Portal

Official NCC NER Region Digital Platform — 15th Tripura Bn NCC

## Tech Stack
- HTML5, CSS3, Vanilla JavaScript
- Font Awesome 6, AOS Animations, Chart.js
- Army Green Theme | Fully Responsive

## Project Structure
```
cadet-connection-portal/
├── index.html, about.html, activities.html, events.html
├── gallery.html, camps.html, login.html, search.html
├── officer/ (CO, ANO, SUO dashboards + management pages)
├── cadet/ (dashboard, profile, id-card, attendance, certificates)
└── assets/ (css/, js/, images/)
```
